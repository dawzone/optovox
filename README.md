# OptoVox

OptoVox is a modern optical vocal compressor built with JUCE.

## Features
- Optical-style program dependent compression
- Tube harmonic coloration (bias control)
- Peak Reduction 0–100 classic control
- HF sidechain emphasis
- Stereo / Mono / M/S modes
- Custom VU + Gain Reduction metering
- Modern clean UI

## Built With
- JUCE Framework
- C++
- VST3 + Standalone

## Status
In active development.
